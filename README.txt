== Installation ==

= Using The WordPress Dashboard =

1. Navigate to "Plugins->Add New" from your dashboard
2. Search for 'Bridgerpay'
3. Click 'Install Now'
4. Activate the plugin

= Uploading via WordPress Dashboard =

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Enable/disable Live mode, depending on the integration.
5. Enter your Cashier key, API key. BridgerPay will provide a separate set API and Cashier Key for each of the environments.

= Shorcode Example =
[bridgercashier theme="dark" price="200" country="CY" currency="EUR" amount_lock="true" button_name="Pay Now" button_mode="spot"]

== Frequently Asked Questions ==

= Can I have multiple payment buttons on a single page? =

Yes, you can have any number of buttons on a single page.

= Can I use it in a WordPress Widgets? =

Yes, you can.

= Can I change the button label? =

Yes, you can specify the "button_text" attribute in the shortcode to customize it.

= Can It be tested before going live? =

Yes, please ask for sandbox credentials and anble Test mode in the plugin configuration

= Where can I see full list of my transactions? =

You will be provided with credentials to Bridger Merchant Admin, where you can fully manage your merchant and transactions.

= How do I lock the amount =

Add the "amount_lock" attribute to the shortcode and set it to true
